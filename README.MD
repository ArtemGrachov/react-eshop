# React e-Shop

Sample application
Developed on Node.js v.18

## How to use in dev mode
1. Run JSON-server
```
npm run db
```
2. Run application
```
npm run dev
```

## How to use in prod mode
1. Run JSON-server
```
npm run db
```
2. Build application
```
npm run build
```
3. Serve application
```
npm run preview
```

## Commands
### Launch dev mode

```
npm run dev
```

### Launch JSON-server

```
npm run db
```

### Compile production build

```
npm run build
```

### Serve production build

```
npm run preview
```
